# 战术模板

按照加成从高到低排序，排名分别为 1 ~ 30，其余人员（替补）排在最后面。分组名称中的"车头：1"
表示加成排名第一的成员为该分组的车头。分组名称中的数字使用加成排名的账号的名称，如排名第一的账号名称是"Alice"
，则分组名称应为"兵器坊（车头：Alice）"。
分组成员根据加成排名依次分配到各个分组中。
分组任务中的数字是分组成员的加成排名，如分组任务为："官渡开放后：1,2抢霹雳车"，排名第一和第二的名称分别为"Alice"和"Bob"
，则任务为："官渡开放后：Alice,Bob抢霹雳车"。

## 战术一

战术：拿下己方所有小粮仓、兵器坊和工匠坊，抢对面上路的一个小粮仓。官渡开放后抢官渡、两个大粮仓和霹雳车。

| 分组名称                  | 成员排名           | 分组任务                                               |
| ------------------------- | ------------------ | ------------------------------------------------------ |
| 兵器坊（车头：1）         | 1,4,8,15,16,21,23  | 4去乌巢，官渡开放后：所有成员抢官渡                    |
| 工匠坊 （车头：2）        | 2,5,10,17,18,22,24 | 5去乌巢，官渡开放后：2,9,10 抢霹雳车,17去黎阳,18去敖仓 |
| 对方上路小粮仓（车头：3） | 3,11,14,19,25      | 3去乌巢，官渡开放后：3,11,14去黎阳，25去敖仓           |
| 我方上路小粮仓（车头：6） | 6,12,13,28         | 官渡开放后：6,12去敖仓，28去兵器坊                     |
| 我方下路小粮仓（车头：7） | 7,20,27,29         | 官渡开放后：7,20去兵器坊，29去工匠坊                   |
| 我方中路小粮仓（车头：9） | 9,26,30            | 官渡开放后：9,26去工匠坊                               |
